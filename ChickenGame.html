Project Overview

The development of this game started with a game jam. My task was to create a clicker game, like Cookie Clicker, within two weeks. I wanted to use this jam not only as an opportunity to improve my coding skills but also to put my art and game design hobbies to the test. Can you discover the secrets of the mysterious Chicken Farm? Download the game to find out.

Technical Details

My first task was to set up and AWS server with Perforce so that I could work with a team if I wanted to. Afterwards, I chose to work in Unreal Engine 5 to learn more about the cool new features.

After researching many of the plugins and systems designs of the Lyra Starter Game, I decided to recreate them in Chicken Game. I found that using the Game Features plugin, Modular Gameplay plugin, and the Gameplay Message Subsystem plugin along with Gameplay Ability System allowed for a very modular and streamlined workflow that helped me finish the game quickly.

In addition to doing all of the coding and design, I created a large portion of the art as well. The game uses Nanite and Lumen, so part of my process was investigating ways to optimize models and lighting while still maintaining a high visual fidelity.
